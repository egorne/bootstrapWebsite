C# Data parsing app
I have designed this app as a part of a university course project with the aim to improve the time-booking system of a datacenter, based on the expected workload, availability of services and energy prices. User would enter the amount of time they needed the services for and if it needed to be done in one timeslot uninterrupted or if it could be separated into multiple half-hour increments. The app would then read data from a .json file with current electricity prices from the electricity providers website, put it against the availability massive and, depending on user input, select and use one of the methods to find the cheapest available time slots, calculate the price and output it to the user, who can then decide if they want to book the time.
